\frame { \frametitle{Hardware Setup \& Performance}
  \begin{itemize}
    \item In single precision, the CaKernel-based CFD code obtains \textbf{$\sim$1 GFLOPS} on
a CPU and \textbf{$\sim$100 GFLOPS} on a NVIDIA GeForce GTX480 GPU.
    \item The scalability tests were performed on up to \textbf{16 nodes} (2 NVIDIA Tesla M2050 GPUs each) 
connected with 40Gbps InfiniBand network.
  \end{itemize}
